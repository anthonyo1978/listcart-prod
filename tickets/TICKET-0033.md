## Goal
Modify the animated cat so it only walks across the screen once per page load. To see the animation again, users must reload the page.

## Requirements
- Change the animation logic to play only once per page session.
- Prevent the cat from looping or repeating automatically.
- Ensure the animation resets on page reload to play again.
- Test on desktop and mobile to confirm the behavior.

## Acceptance Criteria
- [ ] Animated cat walks across the screen only once per page load.
- [ ] Animation does not loop or repeat without page reload.
- [ ] Users can see the animation again after reloading the page.
- [ ] Behavior is consistent and smooth across devices.
