## Goal
Add a large, animated big chubby (Tabby) cat walking across the screen on the website, approximately half the screen size.

## Requirements
- Create or source a large illustration/animation of a big chubby Tabby cat.
- Animate the cat to walk smoothly across the screen horizontally.
- The cat should be approximately half the size of the screen for strong visual presence.
- Position the animation so it doesn't obscure key UI elements but remains prominent.
- Ensure smooth animation and good performance on desktop and mobile.
- Allow the animation to loop or reappear after it finishes.

## Acceptance Criteria
- [ ] Large animated Tabby cat (~half screen size) walks smoothly across the screen.
- [ ] Animation does not interfere with navigation or essential content.
- [ ] Works well and consistently on desktop and mobile browsers.
